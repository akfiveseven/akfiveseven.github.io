EssentialsX

This plugin is used to manage the server economy system and adds helpful quality of life features.

EssentialsX Wiki

  • /spawn

    Teleport to spawn.

  • /bal

    Used to check your account balance.

  • /baltop

    Used to check overall server/player economy.

  • /home <homename>

    Set and warp to a location. Use /sethome <homename> to set homes and /delhome <homename> to delete homes. Members are able to set 3 homes. It costs $25 to warp to a home.

  • /back

    Used to teleport back to your previous location. It costs $100 to use /back.

  • /warp <warpname>

    Used to warp to locations set by admins. Use /warps to see the warp list. It costs $10 to use /warp.

  • /recipe <item|hand>

    Find out the recipe of an item. It costs $5 to use /recipe.

  • /tpa <player>

    Request to teleport to a player. It costs $25 to use /tpa. It also consists of the commands /tpahere, /tpaaccept, /tpadeny.

  • /kit tools

    Get some basic tools, food, and torches. Costs $100 to use /kit tools, and can be used every 30 minutes.

  • /nick <nickname>

    Set a nickname. Costs $100 to use /nick.

  • /realname <player>

    Checks the real name of a player that is using a nickname.

  • /sell hand [amount]

    Sell the item(s) in your hand to the server.

  • /worth

    Check how much an item in your hand is worth.

  • /pay <player> <amount>

    Pay a player some amount of money.

  • /afk

    Broadcast that you are going afk.

  • /mail

    Send mail to players.

ChestShop

Create a personal shop where players can buy from you and/or sell to you.

ChestShop Bukkit Page

Creating Shops

  1. Place a chest on the ground.
  2. Place a certain item in the chest.
  3. Place a sign on it.
  4. On the sign write:
    1. Leave the first line blank.
    2. The second line is the amount of items to buy or sell.
    3. Third line is a combination of buy and sell prices. You need to have a price indicator (like 'B' - price that will apply to items BOUGHT by PLAYERS - or 'S' - the SELL price) near the price - the prices are separated by a colon (:)
    4. Fourth line should be '?'. It will autofill with the item id.

Using Shops

Right-click the sign to buy, left-click to sell.

ChestSort

ChestSort can automatically sort every chest, barrel, etc. after you have used it. Every player can enable or disable this feature if desired with the simple command /sort (or /chestsort). Players can also sort their own inventory using the command /invsort, or automatically using /invsort on|off. You can also use hotkeys (middle-click, double-click, shift-click and shift+right-click) within player inventories, chests etc. to have them sorted, if you do not wish automatic sorting.
